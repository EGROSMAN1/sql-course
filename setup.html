לפני שפותחים את המערכת, ודאו ששני הקבצים הבאים הורדו מה-Google Drive ונמצאים במחשב שלכם:

📄
NORTHWIND(4).sql

שימו לב: הקובץ northwind-data.sql הוא הקובץ החשוב ביותר. ודאו שבחרתם בו ולא בקובץ בעל שם דומה.

🏗️ שלב 1 — הנחת היסודות (Structure)
פתיחת קובץ התשתית ב-MySQL Workbench

פתחו את תוכנת MySQL Workbench.

📂
בתפריט העליון בצד שמאל, לחצו על File ואחר כך בחרו: Open SQL Script...
🖱️
בחלון שנפתח, נווטו לתיקייה שבה שמרתם את הקבצים. בחרו את הקובץ הראשון: NORTHWIND(4).sql
⚡
לאחר בחירת הקובץ, יופיע הקוד על המסך. לחצו על אייקון הברק בסרגל הכלים העליון כדי להריץ את הסקריפט ולבנות את הטבלאות.

* שימו לב: קובץ זה בונה את הטבלאות הריקות. זהו השלד של מסד הנתונים.

✓ סיום שלב 1: הטבלאות נוצרו בהצלחה. כעת המבנה קיים אך הוא ריק מתוכן. השלב הבא יביא את הנתונים.

2
💾 שלב 2 — ייבוא הנתונים (Data)
טעינת קובץ הנתונים לתוך הטבלאות
📂
חזרו על אותו תהליך: File ← Open SQL Script...
🖱️
הפעם בחרו את הקובץ השני: northwind-data.sql — בניגוד לקובץ ההסכמה הקודם
⚡
הקוד החדש יופיע על המסך (במקום הקוד הקודם). לחצו שוב על אייקון הברק כדי להזריק את הנתונים לתוך הטבלאות שיצרנו.

שים לב: שלב זה עלול לקחת מספר שניות — הסקריפט מכניס אלפי שורות נתונים. חכו עד שתראו הודעת הצלחה.

✓
✅ בדיקת איכות — אימות התוצאה
ודאו ש-Northwind מופיע ברשימת ה-Schemas

לאחר שהתהליך הושלם, כך תדעו שהכל עבד:

👈
גשו לחלונית Schemas בצד שמאל ולחצו על כפתור Refresh (כפתור הרענון).
🔍
כעת תראו ברשימה: Northwind — מופיע בכתב מודגש. זה אומר שמסד הנתונים מוכן!

🎉 ברכות! הסביבה מוכנה לעבודה. כעת תוכלו להתחיל לכתוב שאילתות SQL על מסד הנתונים Northwind.

לחצו על Northwind ברשימה כדי לבחור אותו כמסד הנתונים הפעיל, ואז תוכלו להתחיל.

בעיה נפוצה: אם Northwind לא מופיע — ודאו שלחצתם על Refresh, ושהרצתם את שני הקבצים (גם NORTHWIND(4).sql וגם northwind-data.sql).

בדיקה מהירה — שאילתה ראשונה
🚀
בדקו שהכל עובד — הרצת שאילתה ראשונה
אם מקבלים תוצאות — הצלחתם!

פתחו Query Editor חדש ב-Workbench והריצו:

USE Northwind;
SELECT * FROM customers LIMIT 5;

אם מקבלים טבלה עם 5 לקוחות — מסד הנתונים עובד מצוין! 🎉
